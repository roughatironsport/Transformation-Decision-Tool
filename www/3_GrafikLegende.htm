Zur Abbildung: Wenn hohe Anforderungen im Job auf gut entwickelte Skills treffen, so entsteht eine hohe „Kontrolle über das System“. Bei den meisten Mitarbeitern entsteht dabei der Eindruck von Herausforderung aber auch von Zufriedenheit.

Treffen jedoch hohe Anforderungen mit nicht dementsprechend entwickelten Skills zusammen, so resultieren eine geringe Systemkontrolle, Handlungsunsicherheit und Überforderung. Wenn diese Situation länger andauert, so können sich auch erhebliche gesundheitliche Störungen ergeben. Die letztgenannten Effekte gilt es selbstverständlich zu vermeiden.
